The dashboard provides insights into:

- Average price by car manufacturer
- Engine size vs price correlation
- Top vehicle manufacturers
- Fuel type distribution
- Body style distribution
- Average horsepower by manufacturer
- Price range distribution
- MPG (fuel efficiency) analysis

The goal of this project is to transform raw automobile data into **clear, interactive visual insights**.

# 📊 Dashboard Components

### 1️⃣ Average Price by Manufacturer
Displays the **top 10 car manufacturers** with the highest average vehicle prices.

### 2️⃣ Body Style Distribution
Shows the number of vehicles available for each body style:

- Sedan
- Hatchback
- Wagon
- Hardtop
- Convertible

### 3️⃣ Average Horsepower by Manufacturer
Compares horsepower performance across major car brands.

### 4️⃣ Fuel Type Distribution
Analyzes the number of vehicles using:

- Gas
- Diesel

### 5️⃣ Engine Size vs Price Correlation
Scatter plot showing the relationship between:

- Engine size
- Vehicle price

This helps understand how engine capacity affects vehicle cost.

### 6️⃣ Top Manufacturers by Vehicle Count
Shows which companies produce the most vehicles in the dataset.

### 7️⃣ Average MPG by Body Style
Fuel efficiency comparison using:

- City MPG
- Highway MPG

### 8️⃣ Price Range Distribution
Vehicles are categorized into price segments:

- Budget (<10K)
- Mid-Range (10K–20K)
- Premium (20K–30K)
- Luxury (>30K)

# 💡 Key Insights

Some insights derived from the dashboard:

- Luxury brands like **Jaguar and Mercedes-Benz** have the highest average prices.
- **Sedans and hatchbacks** dominate the vehicle distribution.
- Larger engine sizes generally correlate with higher vehicle prices.
- Most vehicles in the dataset fall into the **budget and mid-range categories**.
- Gas-powered vehicles dominate compared to diesel.
